# David MacVicar Portfolio

Knowledge management and technical writing portfolio. Live at [xmacvicar.github.io](https://xmacvicar.github.io).

## Content

- **Case Studies** — end-to-end projects in knowledge governance, tool migrations, and system design
- **Sample Docs** — documentation samples including the Obsidian PKM system guide
- **Writing** — essays on knowledge management, content governance, and technical writing

## Stack

Built with [Docusaurus](https://docusaurus.io) and deployed to GitHub Pages via GitHub Actions.

## Local development

```bash
npm install
npm start
```
