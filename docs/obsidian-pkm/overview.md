# Obsidian PKM Guide: Overview

> **Status:** Sample documentation. This is a complete doc set I wrote about a system I actually use.

A Personal Knowledge Management (PKM) system in Obsidian, built on the PARA framework. This guide is for someone who has installed Obsidian, opened it once, and bounced off because they didn&rsquo;t know where to start.

## What you&rsquo;ll have at the end

- A working vault structured around Projects, Areas, Resources, and Archives
- Templates for the four things you actually need: meeting notes, project briefs, daily logs, reference notes
- A task workflow using the Tasks plugin that surfaces what&rsquo;s due without you having to remember
- Templater automation so new notes show up pre-structured

## What this guide assumes

You have Obsidian installed and you know how to create a note. That&rsquo;s it.

## Time investment

About 90 minutes to read and set up. After that, the system maintains itself if you maintain the habit.
